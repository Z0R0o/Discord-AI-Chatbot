## ✨ Features

- 🤖 @Mention-based replies (no commands required)
- 🧠 Per-user memory system (stored in JSON)
- 🔁 Conversation context persistence
- 💬 Short, clean responses
- ⚙️ Fully modular & easily extensible
- 📁 Auto-saves memory per user in `/data`

## 🧠 How It Works

1. You mention the bot in chat
2. It strips your message, saves context
3. It sends a clean reply back in chat

✅ No commands  
✅ No clutter  
✅ Just clean conversation
